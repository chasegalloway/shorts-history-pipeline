# Privacy Policy — shorts-history-pipeline

_Last updated: July 18, 2026_

**shorts-history-pipeline** is a single-user desktop application built and
operated by its sole user to publish original short-form history videos to
that user's own YouTube channel.

## YouTube API Services

This application uses **YouTube API Services**. By using it, the user agrees
to be bound by the [YouTube Terms of Service](https://www.youtube.com/t/terms).
Google's handling of data is described in the
[Google Privacy Policy](http://www.google.com/policies/privacy).

## What data the application accesses

- The application authenticates via Google OAuth solely to the operator's own
  Google account, with the `youtube.upload` and `yt-analytics.readonly` scopes.
- It uploads videos created locally by the operator to the operator's own
  channel, and reads performance metrics (views, watch time, likes) for those
  same videos only.
- It does **not** access, collect, or process any data belonging to third
  parties or other YouTube users. It does not use search or read public
  YouTube data.

## What data the application stores

- An OAuth token and a local database of the operator's own uploads and their
  performance metrics, stored only on the operator's personal computer.
- No data is shared with, sold to, or transmitted to any third party.
  No cookies or tracking technologies are used.

## Revoking access

The operator can revoke this application's access to their Google account at
any time via the
[Google security settings page](https://security.google.com/settings/security/permissions).
